# 31. Building the Project

### Key Idea

Every software project requires a **build step**.
For web applications (Node.js/React/etc.), the build step:

* Minimizes files
* Optimizes performance
* Prepares artifacts for deployment

---

## 1. Local Build

* Command:

  ```bash
  npm run build
  ```
* Produces an **optimized production build** inside the `build/` folder.
* Build artifacts (like `build/`, `node_modules/`) **should not be committed to Git**.

---

## 2. Jenkins Build Pipeline

### Stage Setup

* Rename stage from `hello` → **Build**
* Use **Docker agent** with Node.js installed:

  ```groovy
  agent {
      docker {
          image 'node:18-alpine'
          reuseNode true
      }
  }
  ```

### Steps

1. **Print Versions** (for troubleshooting):

   ```bash
   node --version
   npm --version
   ```
2. **Install Dependencies**:

    * Use `npm ci` (preferred for CI/CD, faster and reproducible)
      instead of `npm install`.
3. **Run Build**:

   ```bash
   npm run build
   ```
4. **Inspect Workspace**:

    * Before build → no `node_modules/` and no `build/`
    * After build → both appear

---

## 3. Best Practices

* **Test locally first** → then automate in Jenkins
  (saves debugging time between Jenkins vs. tool issues).
* **Print versions** of critical tools (Node, NPM) → helps in debugging later.
* **Warnings during `npm ci`** are fine as long as there are no errors.

---

✅ After this stage, Jenkins produces the **production-ready build folder** that can be deployed.

- Jenkinsfile
````jenkinsfile
pipeline {
    agent any

    stages {
        stage('Build') {
            agent {
                docker {
                    image 'node:18-alpine'
                    args '-u root:root' // run as root to avoid permission issues
                    reuseNode true
                }
            }
            steps {
                sh '''
                    echo "=== Workspace before build ==="
                    ls -lah

                    echo "=== Node & NPM versions ==="
                    node --version
                    npm --version

                    echo "=== Installing dependencies ==="
                    npm ci

                    echo "=== Running build ==="
                    npm run build

                    echo "=== Workspace after build ==="
                    ls -lah
                '''
            }
        }
    }
}
````
