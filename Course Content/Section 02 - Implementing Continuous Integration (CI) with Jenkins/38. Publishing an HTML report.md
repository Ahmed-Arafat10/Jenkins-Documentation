### 🔹 1. Running Playwright with HTML reporter

* Playwright can generate HTML reports with:

  ```bash
  npx playwright test --reporter=html
  ```
* This creates a folder `playwright-report/` containing `index.html`.

---

### 🔹 2. Why report doesn’t display directly in Jenkins

* Jenkins serves workspace files with a **Content Security Policy (CSP)** header for safety.
* This blocks JavaScript and inline styles in your HTML report.
* That’s why opening the HTML report directly from the Jenkins workspace shows a blank page with CSP errors.

---

### 🔹 3. Workaround for CSP (local/test Jenkins)

* You can relax CSP using the **Script Console** in Jenkins:

    * `Manage Jenkins → Script Console`
    * Run a Groovy script that changes Jenkins’ CSP settings.
* ⚠️ Not recommended for production Jenkins at work — only for local/test usage.

- Try to run this script in Jenkins Dashboard > Manage Jenkins > section “Tools and actions” > Script Console:

````
System.setProperty("hudson.model.DirectoryBrowserSupport.CSP", "sandbox allow-scripts;")
````

---

### 🔹 4. Better solution: Publish HTML Report with plugin

* Install the **HTML Publisher Plugin**:

    * `Manage Jenkins → Plugins → Available → HTML Publisher`
* This allows you to expose reports as a clickable link in the Jenkins job page.

---

### 🔹 5. Adding Publisher in Pipeline

* In your `Jenkinsfile`, add a `publishHTML` step.

* To generate the correct syntax:

    * Go to **Pipeline Syntax** in Jenkins job config.
    * Select **publishHTML**.
    * Fill in:

        * `HTML directory to archive`: `playwright-report`
        * `Index page[s]`: `index.html`
        * `Report title`: `Playwright HTML Report`
    * Click **Generate Pipeline Script**.

* Example snippet:

  ```groovy
  post {
      always {
          publishHTML([
              reportDir: 'playwright-report',
              reportFiles: 'index.html',
              reportName: 'Playwright HTML Report'
          ])
      }
  }
  ```

---

### 🔹 6. Result

* After pipeline runs:

    * Jenkins job page shows a sidebar link: **Playwright HTML Report**.
    * Clicking it opens the report directly inside Jenkins UI.
    * No need to browse workspace folders manually.

---

✅ This is the **production-safe way** to view Playwright (or other tool) HTML reports in Jenkins.

> Configuring Content Security Policy: https://www.jenkins.io/doc/book/security/configuring-content-security-policy/

> playwright test reporter: https://playwright.dev/docs/test-reporters


- updated jenkinsfile
````
pipeline {
    agent any

    stages {
        /*

        stage('Build') {
            agent {
                docker {
                    image 'node:18-alpine'
                    reuseNode true
                }
            }
            steps {
                sh '''
                    ls -la
                    node --version
                    npm --version
                    npm ci
                    npm run build
                    ls -la
                '''
            }
        }
        */

        stage('Test') {
            agent {
                docker {
                    image 'node:18-alpine'
                    reuseNode true
                }
            }

            steps {
                sh '''
                    #test -f build/index.html
                    npm test
                '''
            }
        }

        stage('E2E') {
            agent {
                docker {
                    image 'mcr.microsoft.com/playwright:v1.39.0-jammy'
                    reuseNode true
                }
            }

            steps {
                sh '''
                    npm install serve
                    node_modules/.bin/serve -s build &
                    sleep 10
                    npx playwright test --reporter=html
                '''
            }
        }
    }

    post {
        always {
            junit 'jest-results/junit.xml'
            publishHTML([allowMissing: false, alwaysLinkToLastBuild: false, keepAll: false, reportDir: 'playwright-report', reportFiles: 'index.html', reportName: 'Playwright HTML Report', reportTitles: '', useWrapperFileDirectly: true])
        }
    }
}
````
