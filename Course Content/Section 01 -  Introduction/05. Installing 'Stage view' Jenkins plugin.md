# Lecture 5: Installing "Stage View" Jenkins Plugin

## Overview

In recent Jenkins versions, the **Pipeline Stage View** plugin is no longer included in the default set of suggested plugins during installation. Since this plugin will be needed later in the course, this lecture explains how to manually install it and ensure it works properly.

---

## Step-by-Step Installation

### 1. Access Plugin Management

1. Log in to Jenkins.
2. From the left-hand menu, click on **Manage Jenkins**.
3. In the list of management options, select **Manage Plugins**.

### 2. Find the Plugin

1. In the **Available Plugins** tab, use the search bar.
2. Type:

   ```
   Stage View
   ```
3. The result should display the plugin called:
   **Pipeline Stage View**.

### 3. Install the Plugin

1. Select **Pipeline Stage View**.
2. Click **Install without restart** (or **Download now and install after restart**, depending on your Jenkins version).
3. Wait for the installation to finish.

### 4. Restart Jenkins

* After installation, restart Jenkins to finalize setup.
* Jenkins may automatically restart, or you may need to:

    * Manually refresh the page
    * Or restart the container in Docker (if running via Docker).

### 5. Troubleshooting After Restart

* If the browser shows:

  ```
  localhost refused to connect
  ```

  this usually means Jenkins containers are not running.
* Solution:
  Run the same command you used earlier to start Jenkins, for example:

  ```bash
  docker compose up
  ```

---

## Key Takeaways

* The **Pipeline Stage View** plugin is no longer installed by default in modern Jenkins versions.
* To install:

    * Go to **Manage Jenkins → Manage Plugins → Available Plugins**.
    * Search for **Pipeline Stage View** and install it.
* Always restart Jenkins after plugin installation to apply changes.
* If Jenkins doesn’t come back online, ensure Docker containers are running (`docker compose up`).
